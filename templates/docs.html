Добро пожаловать в документацию API для mirea-api.ru

Данное api, созданно исключительно в образовательных целях, оно позволяет получить учебное расписание вуза РТУ МИРЭА.

Для того чтобы использовать API, выполните следующие шаги:

1. Получение доступа

Для доступа к API вам нужно отправить запрос на GET /api/access

2. Примеры запросов

Пример запроса к API для получения данных (расписание группы):

    GET https://mirea-api.ru/api/schedule/group?param=value

3. Ответ API

Ответ API будет в формате JSON.

    {
        "data": "value",
        "status": "success"
    }
